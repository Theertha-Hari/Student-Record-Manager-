# Student Record Manager

A simple **Django project** to practice building CRUD applications for student records.

---

## Features
- Add, view, update, and delete student information
- Uses Django views, templates, and SQLite
- Django admin for backend management

---

## Tech Stack
- Python 3
- Django
- SQLite
- HTML/CSS
